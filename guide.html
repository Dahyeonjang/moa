여행이든 모임이든, 회비와 지출을 기록하고 누가 얼마 주고받을지 한 번에 정산해요. 앱 설치도, 회원가입도 없어요.

3단계면 끝나요

1

모임 만들기

홈에서 이름을 넣고 만들기. 제주 여행, 동창회처럼 모임마다 따로 관리돼요. 끝나면 삭제하면 되고요.

2

그때그때 기록하기

돈 쓸 때마다 지출 · 회비 · 찬조를 골라 금액을 넣어요. 누가 냈는지, 누구랑 나눠 낼지만 체크하면 끝. 영수증 사진도 붙일 수 있어요.

3

정산하고 공유하기

정산 결과 보기를 누르면 누가 누구에게 얼마를 보내면 되는지 딱 나와요. 카톡으로 공유하거나 엑셀로 저장하면 끝.

내보내기 · 가져오기 — 기록 지키기

모여셈은 기록을 내 폰에만 저장해요. 그래서 폰을 바꾸거나 실수로 기록이 사라질 때를 대비해, 정산 화면 맨 아래에 데이터 백업이 있어요.

📤

내보내기

지금까지의 모든 기록(사진 포함)을 파일 하나로 저장해요. 카톡 "나에게 보내기"에 넣어두면 안전한 백업이 돼요.

📥

가져오기

내보낸 파일을 열면 기록이 그대로 되살아나요. 새 폰으로 옮기거나, 컴퓨터에서 이어서 볼 때 써요.

💡 공유하기(카톡·엑셀)와 헷갈리지 마세요. 공유는 사람이 보는 정산 결과를 보내는 거고, 내보내기는 앱이 다시 읽는 백업 파일이에요. 멤버들에게 보낼 땐 공유하기, 내 기록을 지킬 땐 내보내기!

이런 것도 돼요

💱

여러 나라 돈

달러·엔·동 등 통화를 등록하면 환율로 자동 환산. 환율 모르면 🔎 검색 버튼으로 바로 확인해요.

💰

회비·찬조·대신 결제

회비를 걷어도, 누가 대신 카드로 내도, 누가 보태줘도 전부 정확하게 계산돼요.

✈️

인터넷 없이도

홈 화면에 추가하면 비행기 안·해외에서도 열려요. 진짜 앱처럼요.

📊

엑셀로 정리

정산 내역을 표로 저장. 엑셀·구글시트에서 바로 열려요.

📷

영수증 사진

지출마다 사진을 붙여두면 나중에 헷갈릴 일이 없어요.

🔒

내 폰에만 저장

서버에 안 올라가요. 회원가입도 없고, 기록은 오직 내 폰에.

홈 화면에 앱처럼 설치하기

한 번 설치해두면 아이콘으로 바로 열리고, 인터넷 없이도 작동해요.

📱 아이폰 (사파리)
  1. 사파리로 모여셈을 열어요
  2. 아래 공유 버튼(⬆️) → "홈 화면에 추가"
  3. 홈에 🧳 모여셈 아이콘이 생겨요
🤖 안드로이드 (크롬)
  1. 크롬으로 모여셈을 열어요
  2. 오른쪽 위 ⋮ 메뉴 → "홈 화면에 추가"
  3. 홈에 🧳 모여셈 아이콘이 생겨요

자주 묻는 질문

내 기록은 어디에 저장되나요?

오직 내 폰(브라우저)에만 저장돼요. 서버로 보내지 않아서 안전하고, 회원가입도 필요 없어요. 대신 기기를 바꾸면 자동으로 옮겨가지 않으니, 아래 "기기를 바꾸면?"을 참고하세요.

인터넷이 없어도 되나요?

네. 홈 화면에 추가한 뒤에는 비행기 모드·해외 데이터 없이도 앱이 열려요. 환율 검색 버튼만 인터넷이 필요해요(없으면 자동으로 숨겨져요).

기기를 바꾸거나 기록이 사라지면?

정산 화면 아래 데이터 백업 → 내보내기로 파일 하나를 저장해두세요(사진도 함께 담겨요). 새 기기에서 가져오기로 열면 그대로 복원돼요. 여행이 끝나면 한 번 내보내서 카톡 "나에게 보내기"에 넣어두면 안심이에요.

여러 모임을 동시에 쓸 수 있나요?

네. 홈에서 모임을 여러 개 만들 수 있고, 각 모임의 멤버·지출·정산은 완전히 따로 관리돼요.

회비를 안 걷는 모임도 되나요?

물론이에요. 회비 없이 각자 쓴 걸 나누기만 해도 정산이 딱 맞아요.

돈은 얼마예요?

웹앱이라 설치·가입 없이 링크만 있으면 바로 써요.
